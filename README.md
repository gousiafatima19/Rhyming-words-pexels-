🎵 Rhyming Words Generator

A Rhyming Words Generator built using Python and APIs that generates rhyming words and can also fetch related images using the Pexels API.

This project demonstrates creative use of APIs for language and multimedia applications.

🚀 Features

🎵 Generate rhyming words
🖼️ Fetch related images
🔎 Simple interactive usage

🛠️ Tech Stack

Python
Pexels API
Rhyming API
Streamlit

📊 Example Use Cases

Creative writing assistance
Learning API integration
Fun language applications
